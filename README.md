## About

An atmospheric 2D game with an interaction system, dialogue engine, and animated UI. Currently in active development.

## Features

- Interaction system via triggers and IInteractable interface
- Closest interactable object detection by distance
- Dialogue system with TextMeshPro page-by-page display
- Text animation via DOTween (maxVisibleCharacters)
- UI panels with CanvasGroup and animated transitions
- Player movement blocking during dialogues
- ScriptableObjects for text content and settings

## Tech Stack

Unity 2022 | C# | DOTween | TextMeshPro | Physics2D | ScriptableObjects
